HTML Basics

HTML (HyperText Markup Language) is the foundation of web pages.

  • Elements are written with tags like <p> for paragraphs.
  • Most elements require a closing tag, e.g., <p>This is a paragraph.</p>.
  • Some elements, like <img>, are self-closing.
  • Attributes provide additional information about elements, e.g., <a href="https://example.com">Visit Example</a>.

CSS Basics

CSS (Cascading Style Sheets) is used to style web pages.

  • CSS can be written in a separate file and linked in HTML.
  • It styles elements using selectors like p { color: blue; }.
  • Example: body { background-color: #f4f4f4; }

Example Code

Below is a simple HTML & CSS example:

HTML:

<!DOCTYPE html>  <!-- Defines the document type as HTML5 -->
<html>
<head>
    <meta charset="UTF-8">  <!-- Ensures proper text encoding -->
    <title>My First Webpage</title> <!-- Sets the page title -->
    <link rel="stylesheet" href="styles.css">  <!-- Links the external CSS file -->
</head>
<body>
    <h1>Hello, World!</h1>  <!-- Main heading of the page -->
    <p>This is a simple webpage styled with CSS.</p>  <!-- Paragraph of text -->
</body>
</html>

CSS:

/* Simple CSS styling */
body {
    background-color: #f0f0f0; /* Light gray background */
    text-align: center; /* Centers text content */
}

h1 {
    color: #333; /* Dark gray text */
    font-size: 36px; /* Larger heading */
}

p {
    font-size: 18px; /* Readable text size */
    color: #555; /* Medium gray text */
}

How to Run the Example:

  1. Copy and paste the HTML code into a new file named index.html.
  2. Copy and paste the CSS code into a new file named styles.css.
  3. Ensure both files are in the same folder.
  4. Open index.html in a web browser.

Explore Further:

Right-click the webpage and select View Page Source to inspect the HTML.

Use the browser's developer tools (F12 or right-click > Inspect) to explore the CSS styles.
